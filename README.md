# traffic-light-controller-verilog
Traffic light controller using Verilog HDL

# Overview: 

This project implements a priority-based traffic light controller using Verilog HDL and Finite State Machine (FSM) architecture.

# Features:
1. FSM-based traffic control
2. Sequential signal transitions
3. Configurable timing delays
4. Simulation testbench included

# Tools Used
Verilog HDL
Xilinx Vivado

# FSM States

1. RED
2. YELLOW
3. GREEN

# Simulation

The project was validated using simulation waveforms.

Simulation verifies:

Normal traffic sequencing
State transitions
